# Notes

This directory contains learnings, summaries, and documentation captured while working on the project.

## Organization

- Task summaries go here after completing work from TODO.md
- Bug fixes and their root causes
- Architecture decisions and rationale
- Patterns discovered or established

## Format

Each note should include:
- **Date**: When the work was completed
- **Context**: What problem was being solved
- **Summary**: What was done and why
- **Learnings**: Anything useful for future reference
